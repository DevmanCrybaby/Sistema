# MissaoDoDia - Proof of Concept

Pacote: `in.progress.mission`
Linguagem: Kotlin
Objetivo: app simples que salva uma "missão do dia" e exibe em um widget.

Como abrir:
1. Extraia `MissaoDoDia.zip`.
2. Abra o diretório no Android Studio (File → Open).
3. O Android Studio pode pedir para 'Update Gradle wrapper' — permita para importar o projeto.
4. Rode em um emulador ou dispositivo com API >= 26.

Observações:
- Este é um esqueleto mínimo. Se o Android Studio pedir para atualizar plugins/Gradle, aceite.
- Se desejar, eu posso adicionar suporte a clicar no widget para abrir o app, histórico de missões, ou um botão de edição rápida.
